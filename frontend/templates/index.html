About Birthmark classifier

The Birthmark classifier is a tool that classifies birthmarks into different types. It is made by Oskar Männik and Till Bösch for the Introduction to Data Science course at the University of Tartu.

For building the model, we used pretrained ResNet10 model for the image branch and a simple neural network for the tabular branch.

The dataset used for this project can be found here.
